# Fusion 360 Add-in: AI Parameter Optimizer

This Fusion 360 add-in integrates with the AI-Assisted Parametric Design Automation system to apply AI-optimized parameters directly in Fusion 360.

## Features

1. **Get AI Suggestions** - Get AI-optimized parameter suggestions for your design
2. **Apply AI Parameters** - Apply AI-suggested parameters to your design automatically

## Installation

### Method 1: Install as Fusion 360 Add-in

1. **Copy the add-in folder**:
   - Copy the `fusion360_addin` folder to your Fusion 360 add-ins directory:
     - **Windows**: `%APPDATA%\Autodesk\Autodesk Fusion 360\API\AddIns`
     - **Mac**: `~/Library/Application Support/Autodesk/Autodesk Fusion 360/API/AddIns`

2. **Update the path** in the command files:
   - Edit `ApplyAIParameters.py` and `GetAISuggestions.py`
   - Update the `project_root` path to point to your project directory

3. **Install dependencies**:
   - The add-in needs access to your project's `src` directory
   - Ensure the project dependencies are installed in Fusion 360's Python environment

4. **Enable the add-in**:
   - Open Fusion 360
   - Go to **Utilities** â†’ **Add-Ins** â†’ **Add-Ins**
   - Find "AI Parameter Optimizer" and enable it

## Usage

### Get AI Suggestions

1. Open a Fusion 360 design with user parameters
2. Click **Get AI Suggestions** button (in Modify panel)
3. Enter your design intent (e.g., "Minimize weight while maintaining strength")
4. Select AI provider (OpenAI or Anthropic)
5. Review suggestions and explanation
6. Optionally save to JSON file

### Apply AI Parameters

1. Open a Fusion 360 design with user parameters
2. Click **Apply AI Parameters** button
3. Either:
   - Enter design intent to get AI suggestions, OR
   - Provide path to JSON file with optimized parameters
4. Parameters will be updated automatically
5. Design will regenerate with new values

## Configuration

### Environment Variables

The add-in uses environment variables for AI API keys. Set these in your system or Fusion 360 environment:

- `OPENAI_API_KEY` - Your OpenAI API key
- `ANTHROPIC_API_KEY` - Your Anthropic API key (optional)

### JSON File Format

If using a JSON file for parameters, use this format:

```json
{
  "parameters": {
    "length": 95.0,
    "width": 48.0,
    "thickness": 4.0
  },
  "explanation": "AI explanation here...",
  "design_intent": "Minimize weight"
}
```

## Requirements

- Fusion 360 (version 2.0.4000 or later)

  - `openai` (for OpenAI) or `anthropic` (for Anthropic)
  - `requests`

## Troubleshooting

### "AI optimizer not available"

- Ensure the project's `src` directory is accessible
- Check that dependencies are installed
- Verify the path in the command files is correct

### "No user parameters found"

- Make sure your design has user parameters defined
- User parameters are in **Modify** â†’ **Change Parameters** â†’ **User Parameters**

### "AI optimization failed"

- Check your API keys are set correctly
- Verify you have API credits/quota
- Check network connectivity

## Example Workflow

1. **Create a parametric design** in Fusion 360 with user parameters:
